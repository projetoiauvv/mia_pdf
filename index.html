Conversas WhatsApp → PDF

Gerador de PDF estruturado no estilo WhatsApp

Envie um JSON de conversa e gere automaticamente um PDF com as mensagens em ordem cronológica, cabeçalho e rótulos de Cliente e Atendente definidos pelo próprio JSON. Envie um JSON de conversa, informe a atendente e gere automaticamente um PDF com as mensagens em ordem cronológica, cabeçalho, rótulos de Cliente e Atendente.

Se ficar em branco, o app tenta detectar automaticamente pelo prefixo “> Nome” nas mensagens enviadas.
A atendente é identificada automaticamente pelos dados do JSON, como o prefixo > Nome em mensagens enviadas.
Aceita o formato exportado com channel: "Whatsapp" e messages, além de listas diretas de mensagens.
Formato de JSON aceito 
[
  {
    "channel": "Whatsapp",
    "messages": [
      {
        "type": "text",
        "text": "Olá, preciso de ajuda",
        "direction": "INCOMING",
        "created_at": "2026-05-12 10:00:00"
      },
      {
        "type": "text",
        "text": "> Alice\n\nOlá! Como posso ajudar?",
        "direction": "OUTGOING",
        "created_at": "2026-05-12 10:01:00"
      }
    ]
  }
]

O app ignora eventos internos como note e note_action, usa direction para separar cliente/atendente e remove automaticamente o prefixo > Nome do texto exibido.
